# Sistema VED

Sistema Descentralizado VED para fortalecer la integridad de los procesos electorales en la Universidad Nacional José María Arguedas.

## Descripción

Sistema web descentralizado basado en Hyperledger Fabric para la gestión de procesos electorales universitarios, garantizando seguridad, transparencia, integridad y auditabilidad del proceso de votación.

## Stack Tecnológico

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- Shadcn/UI

### Backend
- Node.js
- Express.js

### Base de Datos
- PostgreSQL

### Blockchain
- Hyperledger Fabric
- Chaincode (Node.js)
- CouchDB
